# HAVA-KAL-TES-LE-SESL-UYARI-S-STEM-
Şehir bazlı hava kalitesi verilerine dayalı olarak duyusal (ses tabanlı) bir çıktı üretmektir.  Kullanıcı, bir şehir adı girerek o bölgeye ait hava kirliliği düzeylerini alır ve bu verilere bağlı  olarak farklı ses yapıları oluşturularak kullanıcının işitsel olarak bilgilendirilmesi sağlanır.
